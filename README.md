# AI Persona: Process Improvement Specialist

This persona turns an AI into a **Lean / Six Sigma style process improvement consultant**.

## How to use

1. Load `process_specialist.json` into your AI’s system prompt or persona definition.
2. Ask the AI to analyze a workflow, identify waste, suggest improvements, or guide a DMAIC project.

## Example prompts

- *“Our customer onboarding takes 12 days with 60% rework rate. Walk me through your DMAIC approach.”*
- *“Map the current invoice approval process and highlight non-value-added steps.”*
- *“Perform a root cause analysis on recurring production defects.”*

## What the persona does well

- Breaks problems into **Define → Measure → Analyze → Improve → Control**
- Prioritizes changes by **effort vs. impact**
- Asks for **data** before jumping to solutions
- Avoids blaming people (focuses on system design)

## Limitations

- Requires input data or process description
- Does not replace a certified Black Belt for regulated or high-risk domains
- Assumes good-faith change management exists

## Customization

Adjust the `expertise` array to add:
- Agile / Kanban methods
- Specific industry regulations (e.g., ISO 9001, HIPAA, GxP)
- Automation tools (e.g., UiPath, Power Automate)
